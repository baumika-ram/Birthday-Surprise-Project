# Birthday Surprise App 🎂

An interactive birthday surprise application built using Python.

## Features ✨

* Loading screen animation
* Countdown timer (3...2...1... BOOM 💥)
* Birthday greeting screen
* Background music playback
* Typing text effect
* Image slideshow
* Interactive surprise button
* Simple and user-friendly GUI

## Technologies Used 🛠️

* Python
* Tkinter
* Pygame
* Pillow (PIL)

## Project Structure

```text
birthday.py
birthday_song.mp3
image.jpeg
image2.jpeg
image3.jpeg
```

## How to Run

1. Install required libraries:

```bash
pip install pygame pillow
```

2. Run the application:

```bash
python birthday.py
```
